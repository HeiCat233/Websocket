# 个人主页使用说明

## 📸 如何替换头像

### 方法一：直接替换图片文件（推荐）

1. **准备你的头像照片**
   - 建议尺寸：300x300像素或400x400像素
   - 格式：PNG、JPG、JPEG均可
   - 文件名：`avatar.png`（或其他你喜欢的名字）

2. **将头像文件放到 `www/` 目录下**
   ```
   www/
   ├── avatar.png    ← 把你的头像放在这里
   ├── index.html
   ├── about.html
   └── style.css
   ```

3. **重启服务器**（如果已经在运行）
   ```bash
   python src/main.py
   ```

4. **刷新浏览器**查看效果

---

### 方法二：修改HTML中的图片路径

如果你想使用不同的文件名或放在子目录中：

1. **编辑 `www/index.html`**，找到第25行左右：
   ```html
   <img src="/avatar.png" alt="个人头像" class="profile-avatar" ...>
   ```

2. **修改 `src` 属性**为你的图片路径：
   ```html
   <!-- 例如： -->
   <img src="/images/my-photo.jpg" alt="个人头像" class="profile-avatar" ...>
   ```

3. **同样修改 `www/about.html`** 中的头像路径（第24行左右）

## ✏️ 如何修改个人信息

### 修改姓名和联系方式

编辑 `www/index.html` 文件，找到"个人名片"部分（约第30-38行）：

```html
<section class="card name-card">
    <h2>张三</h2>  <!-- 改成你的名字 -->
    <p class="subtitle">大学生 | 热爱生活 | 喜欢探索</p>  <!-- 改成你的标签 -->
    <div class="contact-info">
        <span class="contact-item">📧 zhangsan@email.com</span>  <!-- 改成你的邮箱 -->
        <span class="contact-item">📱 138-0000-0000</span>  <!-- 改成你的电话 -->
        <span class="contact-item">📍 北京</span>  <!-- 改成你的城市 -->
    </div>
</section>
```

## 🎨 如何修改样式

如果想改变颜色、字体大小等，编辑 `www/style.css` 文件。

### 修改主题颜色

找到 `.name-card` 样式（约第450行）：

```css
.name-card {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);  /* 紫色渐变 */
}
```

可以改成其他颜色：
```css
/* 蓝色渐变 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 橙色渐变 */
background: linear-gradient(135deg, #ff9a56 0%, #ff6b6b 100%);

/* 绿色渐变 */
background: linear-gradient(135deg, #56ab2f 0%, #a8e063 100%);
```
